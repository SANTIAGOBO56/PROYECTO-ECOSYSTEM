Neo4j

¿Qué es Neo4j?

Neo4j es un software libre de base de datos orientada a grafos, implementado en Java. Es un motor de persistencia embebido, basado en disco, completamente transaccional, que almacena datos estructurados en grafos en lugar de en tablas. Está licenciado bajo AGPL v3 y licencia comercial.

Operaciones CRUD en Neo4j

Crear (INSERT):

CREATE (n:Persona {col1: val1, col2: val2});

Leer (SELECT):

MATCH (n:Persona {col1: val1}) RETURN n.col1, n.col2;

Actualizar (UPDATE):

MATCH (n:Persona {col1: val1}) SET n.col2 = nuevo_valor;

Eliminar (DELETE):

MATCH (n:Persona {col1: val1}) DELETE n;
