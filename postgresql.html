PostgreSQL

¿Qué es PostgreSQL?

PostgreSQL es un sistema de gestión de bases de datos relacionales (RDBMS) de código abierto, conocido por su estabilidad, fiabilidad y avanzada funcionalidad. Permite almacenar y gestionar datos tanto relacionales (SQL) como no relacionales (JSON). Es popular para aplicaciones web, móviles y geoespaciales, y se distingue por su comunidad activa y soporte a estándares abiertos.

  • Código Abierto: gratuito, sin costos de licencia.
  • Base de datos relacional y no relacional.
  • Fiabilidad y Resiliencia.
  • Compatibilidad con ACID.
  • Escalabilidad.
  • Funciones Avanzadas.
  • Comunidad de Desarrollo Activa.

Operaciones CRUD en PostgreSQL

CREATE (INSERT)

INSERT INTO productos (nombre, description)
VALUES ('Teclado Mecánico', 'Teclado retroiluminado con switches azules y diseño ergonómico.');

READ (SELECT)

SELECT * FROM productos WHERE nombre = 'Teclado Mecánico';

UPDATE

UPDATE productos
SET description = 'Teclado mecánico de alto rendimiento con switches rojos y retroiluminación RGB.'
WHERE nombre = 'Teclado Mecánico';

DELETE

DELETE FROM productos
WHERE nombre = 'Teclado Mecánico';
