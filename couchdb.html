CouchDB

¿Qué es CouchDB?

CouchDB es una base de datos de documentos NoSQL de código abierto, desarrollada por la Fundación Apache Software Foundation, que utiliza documentos JSON para almacenar datos sin una estructura fija y puede ser accesible vía web mediante HTTP. Se destaca por su arquitectura distribuida, escalabilidad, replicación de datos y tolerancia a fallos, facilitando el desarrollo de aplicaciones web y móviles modernas al permitir la sincronización de datos incluso cuando están offline.

Operaciones CRUD en CouchDB

Crear (INSERT):
PUT /basedatos/mi_doc_id { "col1": "val1", "col2": "val2" }

Leer (SELECT):
GET /basedatos/mi_doc_id

Actualizar (UPDATE):
PUT /basedatos/mi_doc_id?rev=1-xxx { "col1": "val1", "col2": "nuevo_valor" }

Eliminar (DELETE):
DELETE /basedatos/mi_doc_id?rev=1-xxx
