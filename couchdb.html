Couch

es una base de datos de documentos NoSQL de código abierto, desarrollada por la Fundación Apache Software Foundation, que utiliza documentos JSON para almacenar datos sin una estructura fija y puede ser accesible vía web mediante HTTP. Se destaca por su arquitectura distribuida, escalabilidad, replicación de datos y tolerancia a fallos, facilitando el desarrollo de aplicaciones web y móviles modernas al permitir la sincronización de datos incluso cuando están offline.
