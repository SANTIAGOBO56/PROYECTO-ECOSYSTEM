Cassandra

¿Qué es Cassandra?

Se trata de un software NoSQL distribuido y basado en un modelo de almacenamiento «tabular», de código abierto que está escrito en Java. Permite grandes volúmenes de datos en forma distribuida. Por ejemplo, lo usa X para su plataforma. Su objetivo principal es la escalabilidad lineal y la disponibilidad. La arquitectura distribudores de Cassandra está basada en una serie de nodos iguales que se comunican con un protocolo P2P con lo que la redundancia es máxima. Esta soportada por la Apache Software Fundation. Cassandra se basa en una arquitectura distribuida que permite replicar los datos en múltiples nodos, proporcionando tolerancia a fallos y garantizando que los datos permanezcan disponibles incluso en caso de fallos de hardware o de red.

Operaciones CRUD en Cassandra

Crear (INSERT): INSERT INTO tabla (col1, col2) VALUES (val1, val2);

Leer (SELECT): SELECT col1, col2 FROM tabla WHERE col1 = val1;

Actualizar (UPDATE): UPDATE tabla SET col2 = nuevo_valor WHERE col1 = val1;

Eliminar (DELETE): DELETE FROM tabla WHERE col1 = val1;
