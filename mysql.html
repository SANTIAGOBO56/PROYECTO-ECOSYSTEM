MySQL

¿Qué es MySQL?

MySQL es un sistema de gestión de bases de datos relacional de código abierto, rápido y confiable. Permite almacenar, organizar y acceder a datos de forma eficiente mediante el lenguaje SQL.

Es ampliamente utilizado en aplicaciones web, gracias a su integración con tecnologías como PHP, y forma parte de la popular pila LAMP (Linux, Apache, MySQL, PHP).

¿MySQL es Relacional o No Relacional?

MySQL es una base de datos relacional (RDBMS). Organiza los datos en tablas estructuradas con filas y columnas, y permite gestionarlos mediante SQL. A diferencia de las bases de datos NoSQL, que no usan tablas, MySQL se centra en la consistencia y en relaciones bien definidas entre los datos.

Operaciones CRUD en MySQL

Crear (INSERT): INSERT INTO tabla (col1, col2) VALUES (val1, val2);

Leer (SELECT): SELECT col1, col2 FROM tabla WHERE condicion;

Actualizar (UPDATE): UPDATE tabla SET col1 = nuevo_valor WHERE condicion;

Eliminar (DELETE): DELETE FROM tabla WHERE condicion;
