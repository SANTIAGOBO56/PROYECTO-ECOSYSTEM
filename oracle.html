Oracle

¿Qué es Oracle?

Oracle es una empresa tecnológica multinacional líder, y su producto estrella, Oracle Database, es un sistema de gestión de bases de datos relacionales para almacenar, organizar y gestionar grandes volúmenes de información de manera segura, escalable y con alta disponibilidad. Ofrece soluciones integradas en la nube, como infraestructura (OCI) y aplicaciones empresariales (ERP, CRM, etc.).

Ejemplos de operaciones CRUD en Oracle

INSERTAR

INSERT INTO usuarios (nombre_usuario) VALUES ('Juan & Pérez');

SELECT (Leer)

SELECT * FROM productos WHERE nombre_producto LIKE '%Televisor 4K%';

Actualizar (UPDATE)

UPDATE contactos SET telefono = '123-456-7890' WHERE id = 1;

DELETE (Borrar)

DELETE FROM ciudades WHERE nombre = 'Nueva York';
-- Si nombre contiene comillas simples: DELETE FROM ciudades WHERE nombre = 'Nueva'' York';
