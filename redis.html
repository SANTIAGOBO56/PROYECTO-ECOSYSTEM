Redis

¿Qué es Redis?

Redis es un motor de base de datos en memoria, basado en almacenamiento de hashes (clave/valor) pero que opcionalmente puede ser usado como una base de datos durable o persistente. Está escrito en ANSI C y es distribuido bajo RSALv2 y SSP.

Operaciones CRUD en Redis

Crear (INSERT):

SET col1 val1

Leer (SELECT):

GET col1

Actualizar (UPDATE):

SET col1 nuevo_valor

Eliminar (DELETE):

DEL col1
