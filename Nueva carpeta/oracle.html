Oracle

es el nombre de una empresa tecnológica multinacional líder y su producto estrella, Oracle Database, que es un sistema de gestión de bases de datos relacionales para almacenar, organizar y gestionar grandes volúmenes de información de manera segura, escalable y con alta disponibilidad. La compañía ofrece, además, soluciones integradas en la nube, como la infraestructura (OCI) y aplicaciones empresariales (ERP, CRM, etc.) para diversos sectores.

Operaciones CRUD en Oracle

🟢 CREATE – Crear

¿Qué hace? Inserta nuevos datos en una tabla.

INSERT INTO usuarios (nombre, email)
VALUES ('Ana', 'ana@mail.com');

Insertar múltiples registros:

INSERT ALL
INTO usuarios (nombre, email) VALUES ('Luis', 'luis@mail.com')
INTO usuarios (nombre, email) VALUES ('María', 'maria@mail.com')
SELECT * FROM dual;

🔵 READ – Leer

¿Qué hace? Consulta datos existentes.

SELECT * FROM usuarios;

Limitar resultados:

SELECT * FROM usuarios FETCH FIRST 5 ROWS ONLY;

🟡 UPDATE – Actualizar

¿Qué hace? Modifica registros existentes.

UPDATE usuarios
SET email = 'nuevo@mail.com'
WHERE id = 1;

🔴 DELETE – Eliminar

¿Qué hace? Borra registros.

DELETE FROM usuarios
WHERE id = 1;

Eliminar todos los registros rápidamente:

TRUNCATE TABLE usuarios;
