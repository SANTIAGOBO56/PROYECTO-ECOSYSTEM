Que es MongoDB

MongoDB es un sistema de gestión de bases de datos (DBMS) no relacional de código abierto que emplea documentos flexibles en lugar de tablas y filas para procesar y almacenar diversas formas de datos. Como una solución de base de datos NoSQL, MongoDB no requiere un sistema de administración de bases de datos relacional (RDBMS) por lo que proporciona un modelo de almacenamiento de datos elástico que permite a los usuarios almacenar y consultar tipos de datos multivariados con facilidad. Esto no sólo simplifica la administración de bases de datos para los desarrolladores, sino que también crea un entorno altamente escalable para aplicaciones y servicios multiplataforma

MongoDB es Relacional o no Relacional

MongoDB es una base de datos no relacional (o NoSQL) que almacena datos en formato de documentos flexibles en lugar de tablas y filas, lo que permite una gran flexibilidad para datos estructurados, semiestructurados Y no estructurados, y una fácil escalabilidad. A diferencia de las bases de datos relacionales que usan esquemas fijos, MongoDB utiliza un modelo de datos orientado a documentos, como el formato BSON (una forma binaria de JSON).
