MongoDB

¿Qué es MongoDB?

MongoDB es un sistema de gestión de bases de datos NoSQL de código abierto que emplea documentos flexibles en lugar de tablas y filas para procesar y almacenar diversas formas de datos. Permite un modelo de almacenamiento elástico para datos multivariados, simplifica la administración y es altamente escalable para aplicaciones multiplataforma.

Operaciones CRUD en MongoDB

Crear colección (equivalente a tabla):

db.createCollection("Productos")

Insertar documento:

db.Productos.insertOne({nombre: "Laptop Lenovo", precio: 2500.00, stock: 10})
db.Productos.insertOne({nombre: "Mouse Inalámbrico", precio: 50.00, stock: 30})

Leer (Buscar):

db.Productos.find({})

Actualizar:

db.Productos.updateOne({id: 1}, {$set: {precio: 2400.00, stock: 8}})

Eliminar:

db.Productos.deleteOne({id: 2})
