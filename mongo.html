MongoDB

MongoDB es un sistema de gestión de bases de datos NoSQL orientado a documentos y de código abierto. A diferencia de las bases de datos relacionales, almacena la información en documentos BSON (similares a JSON) dentro de colecciones, lo que permite mayor flexibilidad y escalabilidad.

Este gestor es ampliamente utilizado en aplicaciones modernas gracias a su capacidad de manejar grandes volúmenes de datos, su fácil integración con la nube y su eficiencia en entornos distribuidos.

Operaciones CRUD en MongoDB

Las operaciones CRUD (Create, Read, Update, Delete) permiten gestionar los datos en colecciones. A continuación, se muestran ejemplos prácticos:

🟢 CREATE – Crear

Inserta un nuevo documento en una colección.


db.usuarios.insertOne(
  { nombre: "Ana", email: "ana@mail.com" }
);

🔵 READ – Leer

Consulta uno o varios documentos de la colección.


db.usuarios.find(); // Muestra todos los documentos

db.usuarios.find(
  { nombre: "Ana" } // Filtra por nombre
);

🟡 UPDATE – Actualizar

Modifica un documento existente.


db.usuarios.updateOne(
  { nombre: "Ana" }, 
  { $set: { email: "nuevo@mail.com" } }
);

🔴 DELETE – Eliminar

Elimina un documento de la colección.


db.usuarios.deleteOne(
  { nombre: "Ana" }
);
