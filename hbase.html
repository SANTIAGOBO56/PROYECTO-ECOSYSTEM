Apache HBase

Resumen Breve

Apache HBase es una base de datos NoSQL distribuida, de código abierto, diseñada para manejar grandes volúmenes de datos en tiempo real sobre Hadoop y el sistema de archivos HDFS. Está inspirada en el modelo de Google Bigtable. Es ideal para almacenar datos masivos no estructurados con escalabilidad horizontal.

📌 Operaciones CRUD
# Create (Insertar un registro)
put 'productos', 'p1', 'info:nombre', 'Laptop Lenovo'
put 'productos', 'p1', 'info:precio', '2500.00'
put 'productos', 'p1', 'info:stock', '10'

# Read (Consultar)
get 'productos', 'p1'
scan 'productos'

# Update (Actualizar - se hace con otro put sobre la misma clave)
put 'productos', 'p1', 'info:precio', '2400.00'
put 'productos', 'p1', 'info:stock', '15'

# Delete (Eliminar)
delete 'productos', 'p1', 'info:stock'
deleteall 'productos', 'p1'
