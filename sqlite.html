¿Qué es SQLite?

SQLite es un sistema de gestión de bases de datos relacional ligero y de código abierto, implementado como una biblioteca en lenguaje C. A diferencia de otros sistemas, no requiere un servidor independiente, lo que lo hace muy rápido, portátil y fácil de usar.

Es ampliamente utilizado en aplicaciones móviles, navegadores web y dispositivos embebidos gracias a su simplicidad y a que almacena toda la base de datos en un solo archivo.

Características principales

  • No necesita configuración ni instalación de servidor.
  • Almacena toda la base de datos en un único archivo.
  • Compatible con gran parte del estándar SQL-92.
  • Excelente para proyectos pequeños y medianos.
  • Muy eficiente en consumo de memoria.
  • Soporta transacciones ACID.

Ejemplo de operaciones CRUD en SQLite

Crear tabla:

CREATE TABLE Usuarios (
  id INTEGER PRIMARY KEY AUTOINCREMENT,
  nombre TEXT NOT NULL,
  correo VARCHAR(100) UNIQUE
);

Insertar registro:

INSERT INTO Usuarios (nombre, correo) 
VALUES ('Ana López', 'ana.lopez@example.com');

Actualizar registro:

UPDATE Usuarios 
SET correo = 'ana.nueva@example.com' 
WHERE id = 1;

Consultar datos:

SELECT nombre, correo 
FROM Usuarios 
WHERE nombre LIKE 'Ana%';

Eliminar registro:

DELETE FROM Usuarios 
WHERE id = 1;
